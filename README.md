#  CodeAlpha Machine Learning Tasks

##  Overview

This repository contains a collection of Machine Learning projects completed as part of the **CodeAlpha Internship**.

Each project focuses on solving real-world problems using different machine learning techniques such as classification and regression.

---

##  Projects Included

###  1. Iris Flower Classification

* Type: Classification
* Goal: Predict the species of Iris flowers based on measurements
* Algorithm Used: K-Nearest Neighbors (KNN)
* Key Learning: Basics of supervised classification

---

###  2. Car Price Prediction

* Type: Regression
* Goal: Predict the selling price of cars based on features like mileage, engine, and fuel type
* Algorithm Used: Linear Regression
* Key Learning: Data preprocessing, feature engineering, regression modeling

---

###  3. Sales Prediction

* Type: Regression
* Goal: Predict sales based on advertising spend (TV, Radio, Newspaper)
* Algorithm Used: Linear Regression
* Key Learning: Business insights, impact analysis, data visualization
